La durée de ce test est 180min

1. Javascript - 60min
* static/js/app.js

La fonction searchProducts, prend comme arguments une liste de produits, objet de recherche (critères);
Vous devez développer le contenu de la fonction a ce qu'elle retourne les produits qui correspondent aux critères données.

Les données (produits) sont stocké dans "fixtures";

La recherche se fait par:

 - Nom du produit, ex. si on saisi "sou", le produit "sourie" soit apparaître;
 - Prix, ex. si on saisi "100" tous les produits ayant un prix inférieur ou égale à "100" sont corrects;

On peut chercher par Nom, Prix ou bien les deux à la fois.

2. HTML - 45min
* index.html

Proposez une interface de recherche:
 - Un champ text pour un nom de produit recherché;
 - Un champ number pour saisir un prix;
 - Tableau de résultat;

3. Javascript (app) - 45min
* static/js/app.js ( en bas du script )

Quand un utilisateur commence à remplir les champs de recherche, le tableau se met automatiquement à jour avec les résultats trouvés;

4. CSS - 30min
* static/css/style.css

Proposez un style à votre interface, ne pas utilisez bootstrap ou autre framework css.
